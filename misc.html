2021: Musicographics
Musicographics is a graphical music composition tool that was developed together with a musician and architect Ali Murat Cengiz. It's fully written in Unity. It supports loading audio samples, laying out 2D maps and orchestrating it with different instruments, and saving your compositions. The end result is a musical experience that might also resemble underground map of a city.
2018: Turkish Music To Jazz
Worked on the development of the Turkish Language to Jazz Music project for one of the biggest banks in Turkey, Akbank. What this project achieves is that it takes a text in Turkish language as input. The text is analyzed linguistically and turned into a rhythmical structure, which in turn is turned into randomly generated jazz tune using some configurable parameters and rules. My role: Development of the language parsing and music generation methods. Used technologies: JS/Node.js, midi generation plugins & samples
