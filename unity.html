2026 - ECS Survivors - ECS Tech Demo/Showcase in Unity 6
A weekend project exploring Unity 6 ECS/DOTS. A Vampire Survivors-style demo featuring 1000+ simultaneous enemies, Burst-compiled gameplay systems, spatial hash broadphase collision, sprite sheet animation via material property overrides, and shader-driven per-entity health bars. Built to evaluate ECS performance characteristics and developer ergonomics for large-scale gameplay simulations. Link to GitHub
2025 - Room Squared - Sokoban-like Puzzle Game Prototype in Unity
A sokoban-like puzzle game prototype made in Unity 6. Rooms appear as the player builds them using the building boxes. Slide the boxes around to unlock rooms and doors, try to find the secrets and get to the exit! Link to GitHub
2021: Musicographics
Musicographics is a graphical music composition tool that was developed together with a musician and architect Ali Murat Cengiz. It's fully written in Unity. It supports loading audio samples, laying out 2D maps and orchestrating it with different instruments, and saving your compositions. The end result is a musical experience that might also resemble underground map of a city.
